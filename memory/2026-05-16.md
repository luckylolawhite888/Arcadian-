2. Ghost 5s delay fix = remove external cover_image in DB
3. Cloudflare auth: use X-Auth-Email + X-Auth-Key (legacy), not Bearer token
4. SSH via heredoc for complex commands (avoid single-quote escaping hell)
5. nginx: `location = /path` for exact file, `location /path/` for path prefix
6. Never spell magic word — refer only as "the word"
