## 3D Office Embed — Plan for Scarlett Dashboard

### Decision
Approach B: Port the AI Office (Three.js scene) from a Vite/React setup into a standalone JS module that loads inside the existing single-file dashboard HTML. No React, no build step.

### Key Findings
- **OfficeScene.js** is ~1400 lines, entirely procedural geometry (no 3D models). Exports a single function `createOfficeScene(container, callbacks)`.
- **AGENTS array** defines 8 agents (name, color, role, colors). Fully customisable in code.
- **Dependencies:** Three.js only (~500KB gzipped). The scene has no other deps.
- **Mounting:** `createOfficeScene(div)` returns `{setTime, toggleCyberpunk, isCyberpunk, destroy}`.
- **Screen rendering**: Each agent's monitor canvas can be redrawn independently via `screenData` array.
- **Current state**: Agents walk around randomly, talk at water cooler, attend meetings. Shift system (arrival 8am, depart 6pm).
