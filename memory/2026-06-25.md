## Key Lessons
- **Demo first, pitch second** — always let the product speak
- **Two separate interfaces** — sign-up for installers, admin for GPM team. Never show dashboards to people who don't need them.
- **Subdomain setup is fast:** Cloudflare DNS + nginx vhost + scp = ~10 minutes
- **Maya's model is smart:** separate server + accounts per client, Lola builds bot on client infra, Lola is bot overseer. Clean separation, no shared risk.
- **Pricing structure:** £300/mo + 5% gives recurring baseline + upside alignment
- **Commercial unlock:** Same bot architecture, different target list — the system scales to bigger fish
