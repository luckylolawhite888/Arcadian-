## 🧠 Key Learnings
- **Always verify the deployed file** matches the local file — SFTP can silently fail or upload bad state
- **The file got wiped to 0 bytes** when a `fs.writeFileSync` ran after all replace() calls failed silently. Solution: restore from server immediately
- **Express server endpoints are rich** — it has /api/scarlett/find-leads, /api/enrich, /api/voice, etc. The frontend just doesn't call them properly yet
- **Replace() in Node returns the original string if no match** — always check result !== original
- **Every API call needs x-access-code** — server middleware on port 3100 checks it
