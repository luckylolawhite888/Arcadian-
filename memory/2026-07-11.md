### MCS Check — Fixed
- **Problem:** admin-ajax approach returned `0` (no handler registered), wp-json routes returned 404
- **Solution:** Built page scraper that splits HTML by `class="modal-overlay"`, parses modal sections for installer name (`<h2>`), cert#, email (`mailto:`), phone (`tel:`), address, technologies (from `<img alt>`), regions, BUS registration status
- Data source: `mcscertified.com/find-an-installer/?page=1&search=X`
- Returns full structured data for 12 installers per page

### EPC Check — Fixed
- **Problem:** HTML parser didn't match actual GOV.UK page structure
- **Solution:** Uses `govuk-table` class-based parser matching the actual HTML (`<th scope="row" class="govuk-table__header">` for address, `<td class="govuk-table__cell">` for rating)
- URL: `find-energy-certificate.service.gov.uk/find-a-certificate/search-by-postcode?postcode=X`
- Returns total, A-G breakdown, target % (D-G), recent certificates

### MCS Check Scraper — Details
- Scrapes server-rendered HTML from MCS website
- Splits by `class="modal-overlay"` to get individual installer cards
- Regex extraction for: name (h2), cert number (span.pl-1), email (mailto: href), phone (tel: href), address (Address: p tag), technologies (img[alt] in modal), regions, BUS status
- Returns JSON array with rich installer data

### EPC Check Scraper — Details
- Scrapes `find-energy-certificate.service.gov.uk/find-a-certificate/search-by-postcode`
- Parse using `govuk-table epb-search-results` class split
- Row structure: `<th scope="row" class="govuk-table__header"><a href="/energy-certificate/...">ADDRESS</a></th>` + `<td class="govuk-table__cell">RATING</td>`
- Returns A-G breakdown, target % (D-G = solar opportunity)
