**What was fixed:**
1. **Login page broken** — `</div>` etc inside `<script>` tags needed `<\/div>` escaping (HTML parser was ending script early). Fixed by removing backslash escapes entirely (modern HTML5 parser only ends script on `</script>`, not `</div>`)
2. **Emoji rendering** — `\U0001f4dd` (Python-style) displayed literally. Replaced with actual emoji characters in JS source
3. **Double backslash** — `<\\\\/div>` in Python template created `\\/div>` in file (double backslash → escaped backslash). Fixed to output `</div>` (no backslash)
4. **Supabase PATCH fails** — `sbQuery("PATCH", ...)` didn't pass WHERE clause → Supabase requires `id=eq.VALUE` query param for UPDATE. Fixed by direct fetch with proper URL filter
