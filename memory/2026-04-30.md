## 💡 CRITICAL LESSONS
1. **Config structure matters:** `channels.telegram.groups` is a Record `{ "chat_id": {} }`, NOT an Array `[...]`
2. **SSH keys must be stored in a bind-mounted volume** — the container filesystem dies with the container
3. **Don't kill your own PID 15** without verifying the config has valid JSON first
4. **Two gateways = chaos** — always check for stale processes after a restart
5. **Backup the SSH key on the server itself** — so it survives my container being rebuilt
