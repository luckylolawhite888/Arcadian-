# Arcadian Media - Cyberpunk Morning Briefing Dashboard

A standalone cyberpunk-themed dashboard for daily morning briefings, accessible as a Telegram Web App.

## Features
- 🎨 Orange/black cyberpunk theme with glitch effects
- 📰 Live news ticker at the top
- 🚨 Breaking news alerts (flashing)
- ⚽ Sports news section
- 🔮 Daily horoscopes
- 📊 All 10 briefing sections
- ⏰ Live updating clock
- 🔄 Interactive buttons

## Deployment
This site is deployed via GitHub Pages and accessible at:
`https://lolawhite888.github.io/arcadian-media/`

## Telegram Integration
This dashboard is configured as a Telegram Web App. When you tap the "📊 Open Dashboard" button in your morning briefing, this dashboard opens directly inside Telegram.

## Technical Details
- Single HTML file with embedded CSS/JS
- No external dependencies (except Font Awesome CDN)
- Responsive design for mobile & desktop
- Works offline after initial load

## Updates
The dashboard is updated daily with your morning briefing data via automated scripts.
