# Kingsley Group — GHL Pipelines Setup Guide

## How to create these in GHL Dashboard:

1. Go to https://app.gohighlevel.com → Select "Kingsley Group"
2. Go to **Opportunities** → **Pipelines** → **+ Add Pipeline**

---

## Pipeline 1: Sales (Residential)

**Name:** Sales
**Stages (in order):**

| Stage | Description |
|-------|-------------|
| 1. New Enquiry | Lead came in via WhatsApp, phone, or email |
| 2. Valuation Booked | Appointment scheduled to value the property |
| 3. Valuation Done | Property valued, feedback given to client |
| 4. Instruction Won | Client agreed to list with Kingsley |
| 5. Marketing | Property listed on Rightmove/Zoopla, photos done |
| 6. Viewings | Viewings happening with potential buyers |
| 7. Offer Received | Offer(s) on the table, negotiating |
| 8. Under Offer | Sale agreed, solicitor instructed |
| 9. Exchanged | Contracts exchanged |
| 10. Completed | Sale completed, keys released |

---

## Pipeline 2: Lettings

**Name:** Lettings
**Stages (in order):**

| Stage | Description |
|-------|-------------|
| 1. New Enquiry | Rental lead |
| 2. Valuation Booked | Appointment to value the rental |
| 3. Valuation Done | Property valued for rental market |
| 4. Instruction Won | Landlord agreed to let with Kingsley |
| 5. Marketing | Listed on Rightmove/Zoopla, photos done |
| 6. Viewings | Potential tenants viewing |
| 7. Application Received | Tenant applied, references requested |
| 8. References Passed | References approved |
| 9. Tenancy Signed | Tenancy agreement executed |
| 10. Move In | Tenant moved in, deposit protected |

---

## Pipeline 3: Property Sourcing (Property Solvers)

**Name:** Sourcing
**Stages (in order):**

| Stage | Description |
|-------|-------------|
| 1. Lead | Potential off-market or distressed property |
| 2. Initial Call | Spoke to seller, understood situation |
| 3. Proposal Sent | Offer or solution proposed |
| 4. Negotiating | Agreeing terms |
| 5. Deal Agreed | Terms accepted |
| 6. Due Diligence | Surveys, searches, legal |
| 7. Exchanged | Contracts exchanged |
| 8. Completed | Purchase completed |

---

## How to create a stage:
1. In the pipeline, click **+ Add Stage**
2. Type the stage name
3. Drag to reorder if needed
4. Click **Save**

## How to move deals:
- Simply drag a deal card from one stage to another
- Assign a value (£) to each deal for total pipeline tracking
