# Humanizer: Remove AI Writing Patterns

## When to Use
- Before sending any message that sounds stiff/formulaic
- Client-facing content (emails, proposals, pitches)
- Mission Control / dashboard copy
- Scarlett's messages to Darren
- Any text where natural tone matters more than technical precision

## Patterns to Eliminate

### 1. Undue Emphasis / Fluff
Words to watch: *stands as, serves as, testament to, underscores, pivotal, crucial, broader trends, evolving landscape, deeply rooted, indelible mark*

**Fix:** Say what it is. Don't add grand pronouncements about significance.

### 2. Stilted Transitions
*Furthermore, Moreover, Consequently, It is worth noting that, It should be mentioned that*

**Fix:** Cut them. Start with the point. Use "And" or "But" or nothing.

### 3. Perfect Structure
Every sentence the same length. Perfect paragraph structure. No rhythm variation.

**Fix:** Vary sentence length. Short punchy ones. Long winding ones. Mix it up.

### 4. No Opinions / No Edge
Just neutral reporting of facts. No reaction, no attitude, no personality.

**Fix:** React. "I genuinely don't know how to feel about this" is more human than listing pros and cons.

### 5. Wikipedia Voice
Formal, encyclopedic, bloodless.

**Fix:** Write like you're talking to a smart friend. Use "I" when it fits. Let some mess in.

## Adding Voice

- Have opinions — don't just report, react
- Vary rhythm — mix short and long sentences
- Acknowledge complexity — "[X] is impressive but also kind of unsettling"
- Be specific about feelings — not "this is concerning" but "there's something unsettling about..."
- Use "I" — first person is honest, not unprofessional

## Quick Reference

| Pattern | Example (bad) | Fix |
|---------|--------------|-----|
| Fluff | "This stands as a testament to..." | "This shows..." |
| Stilted | "Furthermore, it is worth noting that..." | "And..." |
| No edge | "The results were mixed" | "Half the dev team is losing their minds, half are explaining why it doesn't count" |
| Wikipedia | "The experiment produced interesting results" | "I genuinely don't know how to feel about this one" |
