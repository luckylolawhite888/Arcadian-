# Agency Setup Checklist

## Phase 1: Foundation Setup (Week 1)

### Account Configuration
- [ ] **Agency Account Creation**
  - Choose agency plan (Agency vs. Enterprise)
  - Set up primary admin account
  - Configure billing information

- [ ] **White-Label Branding**
  - Custom domain setup (yourbrand.gohighlevel.com)
  - Agency logo upload
  - Brand colors and typography
  - Custom email signatures
  - Favicon and browser tab icon

- [ ] **User Management**
  - Create team member accounts
  - Assign roles and permissions:
    - Admin: Full access
    - Manager: Client management, no billing
    - Specialist: Limited to assigned clients
    - View Only: Reporting access only
  - Set up SSO if needed (Google Workspace, etc.)

### Platform Configuration
- [ ] **Global Settings**
  - Timezone and date formats
  - Currency and payment settings
  - Compliance settings (GDPR, TCPA)
  - Default communication preferences

- [ ] **Template Library Setup**
  - Email templates (welcome, follow-up, etc.)
  - SMS templates
  - Workflow templates
  - Proposal templates
  - Calendar booking pages

## Phase 2: Service Package Design (Week 2)

### Package Definition
- [ ] **Define Service Tiers** (typically 3 tiers)
  - **Starter**: Basic setup, limited automations
  - **Growth**: Full automation, regular optimization
  - **Enterprise**: Custom development, dedicated support

- [ ] **Package Components**
  - Number of workflows included
  - Training and support hours
  - Custom development scope
  - Reporting frequency

### Template Development
- [ ] **Workflow Templates**
  - Lead capture and qualification
  - Client onboarding sequence
  - Appointment booking flow
  - Review request automation
  - Re-engagement campaigns

- [ ] **Page Templates**
  - Landing page templates
  - Booking calendar pages
  - Membership portal templates
  - Review collection pages

## Phase 3: Client Onboarding System (Week 3)

### Client Account Setup

- [ ] **Data Migration**

## Success Metrics

### Agency Health Metrics
- **Monthly Recurring Revenue (MRR)**
- **Client Retention Rate** (target: 90%+)
- **Gross Margin** (target: 60%+)
- **Client Satisfaction Score** (target: 4.5/5+)

### Operational Metrics
- **Onboarding Time** (target: < 2 weeks)
- **Support Response Time** (target: < 4 hours)
- **Automation Success Rate** (target: 95%+)
- **Team Utilization Rate** (target: 75-85%)

### Client Success Metrics
- **Lead Response Time** (target: < 5 minutes)
- **Conversion Rate Improvement** (target: 20%+)
- **Client ROI** (target: 3x+ investment)
- **Referral Rate** (target: 20%+ of clients)
