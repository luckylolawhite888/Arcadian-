# Radio Station Project — Station Control FM

## Concept
Maya's internet radio station. Hybrid approach: **AzuraCast** (self-hosted engine on IONOS) + **Radio.co** (licensing + distribution + app).

## Tech Stack
- **Engine:** AzuraCast (self-hosted on IONOS server via Docker)
- **Distribution:** Radio.co (handles licensing, streaming, mobile apps)
- **Music sourcing:** DJ pools (BPM Supreme / DJcity ~£20-30/mo) + Maya's own DistroKid tracks

## AzuraCast Setup
- Run via Docker on IONOS
- Handles AutoDJ, playlist management, live show scheduling
- Streams to Radio.co for distribution

## Radio.co Benefits
- Handles licensing/royalties reporting
- Provides mobile apps (iOS + Android)
- Distribution to TuneIn, iTunes, etc.
- Professional streaming infrastructure

## The 6 Agents
| Agent | Role |
|-------|------|
| 🎧 **AutoDJ** | Music scheduling, playlists, automated shows |
| 📢 **Marketing** | Social media, promotion, listener growth |
| 💰 **Ad Sales** | Sell airtime to local businesses |
| 🎤 **Slot Sales** | Sell show slots (£10/hr) to aspiring DJs |
| 📊 **Monitor** | Track listeners, performance, uptime |
| 🧠 **Station Manager** | Coordinates everything |

## Out & About Synergy
- **Free radio ads** as sign-up incentive for first 50 Out & About shops
- **Window stickers:** "Find our deals on the Out & About app — as heard on Station Control FM"
- **Bundle pitch:** "£79/mo gets you an app listing AND a radio ad"
- **Cross-promo:** Each shop gets a :30-:60 radio ad included in Growth plan
- **Revenue share:** Ad sales + slot fees supplement subscription revenue

## Revenue Model
| Stream | Source |
