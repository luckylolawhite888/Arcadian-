# Food Buttons System

## What It Is
An interactive menu when Maya mentions hunger — shows themed buttons for food categories.

## File
- `food_buttons.md` — Button definitions and template

## Button Options
| Emoji | Category |
|-------|----------|
| 🍕 | Pizza |
| 🍔 | Burger |
| 🥡 | Chinese |
| 🌮 | Tacos |
| 🍜 | Ramen |
| 🍰 | Dessert |
| 🌙 | Late Night Snacks |

## Trigger
When Maya says she's hungry or asks about food, Lola automatically offers the food buttons menu.

## Workflow
1. Maya mentions hunger
2. Lola shows food buttons
3. Maya clicks a choice
4. Lola shows recommendations with booking/delivery links
